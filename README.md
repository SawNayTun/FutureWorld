# ၂လုံးထီ စီမံခန့်ခွဲမှုစနစ် (2D Lottery Management System)

ဤဆော့ဖ်ဝဲလ်သည် ၂လုံးထီ ဒိုင်များ၊ အလယ်ဒိုင်များနှင့် အေးဂျင့်များအတွက် အရောင်းစာရင်းများကို စီမံခန့်ခွဲရန်၊ ဘဏ္ဍာရေးအခြေအနေများကို ခြေရာခံရန်နှင့် လုပ်ငန်းလည်ပတ်မှုကို ပိုမိုလွယ်ကူစေရန် ဖန်တီးထားသော ခေတ်မီဒက်ရှ်ဘုတ်တစ်ခုဖြစ်ပါသည်။

## အဓိကအင်္ဂါရပ်များ (Features)

*   **သုံးစွဲသူ Role (၃) မျိုး:** အေးဂျင့်၊ အလယ်ဒိုင်၊ ဒိုင်ကြီးဟူ၍ မတူညီသော Role များအတွက် သီးသန့်လုပ်ဆောင်ချက်များ။
*   **စာရင်းသွင်းမှုစနစ်:** မြန်မာ-အင်္ဂလိပ် ၂လုံးအခေါ်အဝေါ်များ (`apu`, `ညီကို`, `12r`) ဖြင့် စာရင်းများကို လျင်မြန်စွာထည့်သွင်းနိုင်ခြင်း။
*   **စာရင်းပြင်ဆင်မှု:** နောက်ဆုံးထည့်သွင်းခဲ့သော စာရင်းများကို အလွယ်တကူ ပြန်လည်ကြည့်ရှု၊ ပြင်ဆင်၊ ဖျက်နိုင်ခြင်း။
*   **ဘဏ္ဍာရေးအကျဉ်းချုပ်:** ကိုင်ငွေ၊ လစ်မစ်ကျော်/ကာသီး၊ ကော်မရှင်များနှင့် အသားတင်အမြတ်တို့ကို အချိန်နှင့်တပြေးညီ တွက်ချက်ပြသပေးခြင်း။
*   **လုံခြုံသော လိုင်စင်စနစ်:** Device ID ကိုအခြေခံထားသော လိုင်စင်စနစ်နှင့် အက်ဒမင်အတွက် စကားဝှက်ဖြင့် ထိန်းချုပ်မှု။
*   **မှတ်တမ်းနှင့် Report များ:** နေ့စဉ်စာရင်းများကို သိမ်းဆည်းနိုင်ပြီး အပတ်စဉ်၊ လစဉ်၊ နှစ်စဉ် အကျဉ်းချုပ် Report များကို ကြည့်ရှုနိုင်ခြင်း။
*   **Backup & Restore:** ဒေတာအားလုံးကို local file အဖြစ် Backup လုပ်နိုင်ပြီး ပြန်လည် Restore လုပ်နိုင်ခြင်း။
*   **Build-less Setup:** `npm install` သို့မဟုတ် build process များမလိုအပ်ဘဲ အလွယ်တကူ run နိုင်ခြင်း။

## နည်းပညာများ (Technology Stack)

*   **Frontend:** Angular (Zoneless)
*   **Styling:** Tailwind CSS (via CDN)
*   **Data Storage:** Browser IndexedDB
*   **Dependencies:** Loaded via Import Maps from CDN (esm.sh)

## Project ကို Run ရန် လမ်းညွှန် (Setup and Running Instructions)

ဤ Project သည် အလွန်ရိုးရှင်းစွာ ဒီဇိုင်းဆွဲထားသောကြောင့် `npm` သို့မဟုတ် အခြား dependency manager များမလိုအပ်ပါ။ အောက်ပါအဆင့်များအတိုင်း အလွယ်တကူ Run နိုင်ပါသည်။

### အဆင့် ၁: Project ကို Download ဆွဲပါ

GitHub repository မှ Project ကို သင်၏ကွန်ပျူတာထဲသို့ Clone လုပ်ပါ သို့မဟုတ် ZIP file အဖြစ် download ဆွဲပါ။

```bash
# ဥပမာ - Git ဖြင့် Clone လုပ်ရန်
git clone <your-repository-url>
```

### အဆင့် ၂: Local Web Server ဖြင့် Run ပါ

လုံခြုံရေးဆိုင်ရာ browser ကန့်သတ်ချက်များကို ရှောင်ရှားရန်၊ `index.html` ဖိုင်ကို တိုက်ရိုက်ဖွင့်မည့်အစား local web server ဖြင့် run ရန် အကြံပြုပါသည်။

1.  သင်၏ Terminal သို့မဟုတ် Command Prompt ကိုဖွင့်ပြီး Project folder ထဲသို့သွားပါ။
    ```bash
    cd path/to/your/project-folder
    ```
2.  သင့်ကွန်ပျူတာတွင် Node.js ထည့်သွင်းထားပါက၊ အောက်ပါ command ကို အသုံးပြု၍ server ကို အလွယ်တကူ run နိုင်ပါသည်။
    ```bash
    npx serve .
    ```
    (အကယ်၍ `serve` ကို မထည့်သွင်းရသေးပါက၊ `npx` က သင့်အား ထည့်သွင်းရန် အလိုအလျောက်မေးမြန်းပါလိမ့်မည်။ `y` ဟုရိုက်၍ အတည်ပြုပါ။)

3.  Server run သွားပါက၊ Terminal တွင် ပြထားသော URL (ဥပမာ: `http://localhost:3000`) ကို browser တွင်ဖွင့်လိုက်ပါ။

### အဆင့် ၃: ပထမဆုံးအကြိမ် အသုံးပြုခြင်း

*   App ကို ပထမဆုံးဖွင့်သည့်အခါ "ဆော့ဖ်ဝဲလ် အသက်သွင်းခြင်း" စာမျက်နှာသို့ ရောက်ရှိပါမည်။
*   **အက်ဒမင် (Admin)** အဖြစ်ဝင်ရောက်ရန်၊ စကားဝှက်အကွက်တွင် default password ဖြစ်သော **`MasterSaiYan`** ကိုရိုက်ထည့်ပြီး "လိုင်စင် အသက်သွင်းမည်" ကိုနှိပ်ပါ။
*   အက်ဒမင် Panel မှတစ်ဆင့် သာမန်အသုံးပြုသူများအတွက် လိုင်စင်ကီးများ ထုတ်ပေးနိုင်ပါသည်။

## ဒေတာသိမ်းဆည်းမှု (Data Persistence)

*   သင်၏ စာရင်းများ၊ အေးဂျင့်များ၊ မှတ်တမ်းများ အစရှိသည့် ဒေတာအားလုံးကို သင်အသုံးပြုနေသော browser ၏ **IndexedDB** တွင်သာ သိမ်းဆည်းထားပါသည်။
*   Browser data (cache, cookies, site data) များကို ရှင်းလင်းလိုက်ပါက **App ရှိ ဒေတာအားလုံး ပျက်စီးသွားမည်** ဖြစ်ပါသည်။
*   ဒေတာမဆုံးရှုံးစေရန် **Backup & Restore** အင်္ဂါရပ်ကို ပုံမှန်အသုံးပြုရန် အထူးအကြံပြုလိုပါသည်။
