အသုံးပြုသူ လမ်းညွှန်

မိတ်ဆက်

ဤဆော့ဖ်ဝဲလ်သည် ၂လုံးထီ ဒိုင်များနှင့် အလယ်ဒိုင်များအတွက် စာရင်းများကို လွယ်ကူတိကျစွာ စီမံခန့်ခွဲနိုင်ရန် ဖန်တီးထားပါသည်။ Mode (၂) မျိုးဖြင့် သင့်လုပ်ငန်းလိုအပ်ချက်အလိုက် အသုံးပြုနိုင်ပါသည်။

လုံခြုံရေးအတွက်၊ သင်သည် application ကိုပိတ်လိုက်တိုင်း (ဥပမာ- browser tab ကိုပိတ်ခြင်း) အလိုအလျောက်ထွက်သွားမည်ဖြစ်ပြီး နောက်တစ်ကြိမ်အသုံးပြုသည့်အခါ လိုင်စင်ကီး/စကားဝှက်ဖြင့် ပြန်လည်ဝင်ရောက်ရန် လိုအပ်ပါသည်။

အထွေထွေ ဆက်တင်များ

  • ငွေကြေး သတ်မှတ်ခြင်း: ပင်မစာမျက်နှာ၏ ဘယ်ဘက်ခြမ်းရှိ ဆက်တင်များတွင် သင်အသုံးပြုလိုသော ငွေကြေး (ကျပ်/ဘတ်/ယွမ်) ကို ရွေးချယ်နိုင်ပါသည်။ ရွေးချယ်လိုက်သော ငွေကြေးသည် app တစ်ခုလုံးရှိ ငွေကြေးပြသမှုများတွင် သက်ရောက်မည်ဖြစ်သည်။
  • Heatmap ကြည့်ရှုခြင်း: "Heatmap ဖွင့်" ခလုတ်ကို နှိပ်၍ ဂဏန်းများ၏ အရောင်းအနေအထားကို အရောင်ဖြင့် ခွဲခြားကြည့်ရှုနိုင်ပါသည်။ (အစိမ်း = နည်း၊ အဝါ = သင့်တင့်၊ လိမ်မော် = များ၊ အနီ = လစ်မစ်ကျော်)။

Mode အလိုက် အသုံးပြုပုံ

၁။ အလယ်ဒိုင် (Middle Bookie)

  • စီမံခန့်ခွဲမှု: ညာဘက်ခြမ်းရှိ "အေးဂျင့်များ စီမံရန်" နှင့် "အပေါ်ဒိုင်များ စီမံရန်" အကွက်များတွင် သင်၏လက်အောက်ခံ အေးဂျင့်များနှင့် သင်ကာသီးလွှဲရမည့် အပေါ်ဒိုင်များကို ၎င်းတို့၏ ကော်မရှင်များနှင့်တကွ ထည့်သွင်းပါ။
  • Default လစ်မစ်: ဂဏန်းတစ်ခုချင်းစီအတွက် သင်အများဆုံးကိုင်မည့် ပမာဏကို သတ်မှတ်ပါ။
  • Inbox မှ စာရင်းလက်ခံခြင်း: ညာဘက်ခြမ်း "Inbox" တွင် လက်ခံရရှိသော စာရင်းများကို Paste ချပါ၊ သက်ဆိုင်ရာ Agent ကိုရွေးချယ်ပြီး "ထည့်မည်" ကိုနှိပ်ပါ။
  • Inbox အသုံးပြုနည်း (အမြန်): Agent စာရင်းများထည့်သွင်းရာတွင်၊ စာရင်း၏ ပထမဆုံးစာကြောင်း၌ `Agent Name:` ဟု ရိုက်ထည့်ပြီး စာရင်းထည့်ပါက သက်ဆိုင်ရာ Agent ကို အလိုအလျောက် ရွေးချယ်ပေးမည်ဖြစ်ပြီး dropdown မှ ရွေးရန်မလိုတော့ပါ။ ဥပမာ: `Maung Maung:\n12r 1000\n...`
  • ကာသီး: သင်၏ Default Limit ကိုကျော်လွန်သွားသော ထိုးကြေးများကို "ကာသီး" အဖြစ် သတ်မှတ်ပြီး ဇယားတွင်ပြသပေးပါမည်။ "ကာသီးစာရင်းများ စီမံရန်" ခလုတ်ဖြင့် အပေါ်ဒိုင်များထံသို့ စာရင်းများကို အလွယ်တကူ ခွဲဝေส่งต่อနိုင်ပါသည်။
  • ဘဏ္ဍာရေး: သင်၏ ကိုင်ငွေ၊ ကာသီး၊ ပေးရမည့်/ရမည့် ကော်မရှင်များနှင့် အသားတင်အမြတ်ကို ညာဘက်ခြမ်း "ဘဏ္ဍာရေး အကျဉ်းချုပ်" တွင် အချိန်နှင့်တပြေးညီ ကြည့်ရှုနိုင်ပါသည်။

၂။ ဒိုင်ကြီး (Main Bookie)

  • အေးဂျင့် စီမံခန့်ခွဲမှု: "အလယ်ဒိုင်" ကဲ့သို့ပင် သင်၏ လက်အောက်ခံအေးဂျင့်များကို စီမံပါ။
  • Default & Individual Limit: Default Limit အပြင်၊ ဂဏန်းတစ်ခုချင်းစီကို သီးသန့် Limit သတ်မှတ်လိုပါက ပင်မဇယားရှိ မည်သည့်ဂဏန်းကိုမဆို Click နှိပ်၍ သတ်မှတ်နိုင်ပါသည်။
  • Risk Management: "Risk စစ်ဆေးမည်" ခလုတ်ကို နှိပ်၍ ထိုးကြေးအများဆုံး ဂဏန်းများနှင့် အကယ်၍ပေါက်ခဲ့ပါက ရှုံးနိုင်မည့်ပမာဏများကို ကြည့်ရှုနိုင်ပါသည်။
  • စာရင်းရှင်းခြင်း: ညာဘက်အပေါ်ထောင့်ရှိ "ပေါက်ဂဏန်း စာရင်းရှင်းရန်" ခလုတ်ကိုနှိပ်၍ ပေါက်ဂဏန်းထည့်သွင်းပြီး လျော်ကြေးငွေ စာရင်းကို အလွယ်တကူ တွက်ချက်နိုင်ပါသည်။

အသံစနစ် (Voice Input)

စာရင်းသွင်းသည့်အကွက်ရှိ မိုက်ကရိုဖုန်းပုံစံ ခလုတ်ကိုနှိပ်၍ အသံဖြင့် စာရင်းသွင်းနိုင်ပါသည်။

  • ဂဏန်းများ: မြန်မာလို (တစ်၊ နှစ်၊ သုံး) သို့မဟုတ် အင်္ဂလိပ်လို (One, Two, Three) ပြောဆိုနိုင်ပါသည်။
  • အခေါ်အဝေါ်များ: အပူး၊ နက္ခတ်၊ ပါဝါ၊ ခွေ၊ အာ (R) စသည့် အခေါ်အဝေါ်များကို အသံဖြင့် ပြောဆိုနိုင်ပါသည်။
  • Command များ: "ပြန်ဖျက်" (Cancel)၊ "ထည့်မယ်" (Save) ဟုပြော၍ ခလုတ်နှိပ်စရာမလိုဘဲ လုပ်ဆောင်နိုင်ပါသည်။

စာရင်းသွင်းခြင်းနှင့် Shortcuts

  • Enter ဖြင့် စာရင်းထည့်သွင်းခြင်း: စာရင်းသွင်းအကွက်တွင် စာရင်းရိုက်ပြီး `Enter` ခေါက်လိုက်သည်နှင့် "ထည့်မည်" ခလုတ်ကို နှိပ်စရာမလိုဘဲ စာရင်းကို ဇယားထဲသို့ တိုက်ရိုက်ထည့်သွင်းပေးပါမည်။
  • Shift + Enter ဖြင့် လိုင်းဆင်းခြင်း: စာရင်းသွင်းအကွက်ထဲတွင် လိုင်းအသစ်ဆင်းလိုပါက `Shift` key နှင့် `Enter` key ကို တွဲနှိပ်ပါ။

အသုံးပြုသူအသစ်အတွက် လိုင်စင်ထုတ်ပေးလိုပါက "Admin Panel" သို့သွားပါ။ "လိုင်စင်ထုတ်ရန်" အပိုင်းတွင် အသုံးပြုသူ၏အမည်၊ သူထံမှရရှိသော `device-id.future2d` ဖိုင်ကိုတင်၍ Device ID နှင့် သက်တမ်းကုန်ဆုံးမည့်ရက်တို့ကို ဖြည့်စွက်ပြီး လိုင်စင်ထုတ်ပေးနိုင်ပါသည်။ ထုတ်လုပ်ပြီး လိုင်စင်ကီးကို အသုံးပြုသူထံသို့ ပေးပို့ပါ။

အသုံးပြုသူ စီမံခန့်ခွဲမှု

Admin Panel ၏ ညာဘက်ခြမ်းတွင် သင်ထုတ်ပေးထားသော အသုံးပြုသူများ၏ စာရင်းကို ကြည့်ရှုနိုင်ပါသည်။ လိုင်စင်ကီးများ Copy ကူးခြင်း၊ အသုံးပြုသူများကို ဖျက်ခြင်းများ ပြုလုပ်နိုင်ပါသည်။

မာစတာကီး (စကားဝှက်) ပြောင်းလဲခြင်း

သင်၏ Admin Account သို့ Login ဝင်သော စကားဝှက်ကို "မာစတာကီး ပြောင်းရန်" အပိုင်းတွင် ပြောင်းလဲနိုင်ပါသည်။ လုံခြုံရေးအတွက် ပုံမှန်ပြောင်းလဲပေးသင့်ပါသည်။
